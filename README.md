Frontend (Next.js): Open http://localhost:3000 in your browser.
Backend (Flask API): The backend will be running on http://127.0.0.1:5328.

1. Run:

```sh
pnpm install
```

2. Run:

```sh
pip install -r requirements.txt
```

3. Run:

```sh
pnpm run dev
```
